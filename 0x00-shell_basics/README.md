A readme file at the root of my directorY
#pwd
Print current working directory

#ls
list the file contents of current directory

#cd .. 
Change the current directory to the home directory

#ls -l
List file contents of current directory in a long format

#ls -la
List file contents incuding hidden files of current directory in a long format

#ls -na
List file contents including hidden files of current directory showing their groupId and UserId numerically

#mkdir /tmp/my_first_directory
Creates a directory

#mv /tmp/betty /tmp/my_first_directory
Move the betty file inside the tmp directory into my_first_directory inside tmp as well

#rm -r /tmp/my_first_directory/betty
Remove the betty file 

#rm -r /tmp/my_first_directory
Remove my_first_directory 

#cd -
Change the working directory to the previous one

#ls -la . ../
Print the long file and hidden file of current working directory and parent working directory

#file /tmp/iamfile/
print the type of file that is the above directory/file


#ln -s /bin/_ls_
Attempt to link the current working directory with the latter directory

#cp -i *.html ../
uses the wildcat to copy all html files in the cuurent directory to the parent directory
